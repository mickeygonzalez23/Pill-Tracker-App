# Hardware Buying Notes

## Short Answer

The MacBook Neo should work for building Did I Take It? Pill Tracker because Apple lists it as a Mac running macOS.

You do not need an M1 processor specifically.

## Good Options

```text
MacBook Neo with A18 Pro
MacBook Air with M1, M2, M3, or M4
MacBook Pro with M1, M2, M3, or M4
Mac mini with M1, M2, M3, or M4
```

## What Matters Most

```text
Runs macOS
Can install Xcode
Can connect to your iPhone
Has enough storage for Xcode and simulators
```

## Storage Recommendation

Choose 512 GB storage if possible.

256 GB can work, but Xcode and iPhone simulators can take a lot of space.

## Memory Recommendation

8 GB memory is acceptable for this app.

16 GB is nicer for long-term development, but not required for this project.

## What To Avoid

Avoid very old Intel MacBooks unless you confirm they can run a recent enough version of macOS and Xcode.

## Cost Reminder

```text
Xcode: free
Private testing on your iPhone: usually free
Apple Developer Program for App Store publishing: $99/year
```
